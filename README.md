# AstrBot 发送BV号获取哔站视频封面图

## 简介

发送BV号获取哔站视频封面

## 配置

本插件无需任何配置。

## 注意事项

*   本插件需要 AstrBot 机器人环境。
*   请确保 AstrBot 机器人已正确安装并配置。
*   无法获取手机从哔站直接分享的视频(就是那种以小程序的形式)
*   无法获取手机从哔站点击按钮“复制链接”的形式(就是点分享后点下面的复制链接)
*   手机只能复制BV号来获取，电脑网页可以直接复制网址或者点分享按钮复制的链接。
*   技术有限，如果有大佬可以改的话万分感激！
